## 输入上下文

### 需求摘要（来自 Discovery 阶段）
为账单导入服务添加前端复盘功能，实现：
1. 复盘配置管理（设置页面）
2. 账单导入后智能提示复盘
3. 复盘报告生成和管理

### 代码库分析结果（来自 Exploration 阶段）
- 项目使用 Vanilla JavaScript + Jinja2 模板
- 前端代码组织清晰，有统一的组件模式
- 设置页面使用侧边栏导航 + 多section布局
- 账单导入页面已有成功提示和Banner模式
- 后端API已完整实现

### 已知后端API
- POST `/api/review/generate` - 生成复盘报告
- POST `/api/review/batch` - 批量生成复盘
- GET `/api/review/config` - 获取复盘配置
- POST `/api/review/config` - 更新复盘配置
- GET `/api/review/preview` - 预览复盘数据

### 技术约束
1. 项目使用原生JavaScript，不使用前端框架
2. 需要在main.py中注册review router
3. 设计需与现有UI风格保持一致
